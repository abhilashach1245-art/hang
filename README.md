# 🎮 Hangman Game (Python)

A simple command-line Hangman game built using Python. Guess the hidden word one letter at a time before you run out of attempts!

## 📌 Features

* Random word selection
* Tracks guessed letters
* Limited number of wrong attempts
* Simple and beginner-friendly code
* Console-based gameplay

## 🛠️ Technologies Used

* Python
* Built-in libraries (`random`)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/hangman-game.git
```

2. Navigate to the project folder:

```bash
cd hangman-game
```

3. Run the Python file:

```bash
python hangman.py
```

## 🎯 How to Play

* The game randomly selects a word.
* You need to guess the word letter by letter.
* Each wrong guess increases your mistake count.
* You lose if you exceed the maximum number of wrong guesses.
* You win if you guess the word correctly before running out of chances.

## 📷 Example

```
Welcome to Hangman!
Guess the word: _ _ _ _ _
Enter a letter: a
Correct guess!
```
